#В данной работе реализован код для шифра Бифида и шифра ADFGX ЯП C++. Ниже представленны все нужные данные для работы с данными шифрами:

--- Шифр Бифида: Шифр Бифида (от лат. bifidus — «разделённый надвое») — классический симметричный шифр подстановки с фракционированием, изобретённый французским криптографом Феликсом Деластеллем в 1901 году. 1. Базовые компоненты Квадрат Полибия (5×5) Основой шифра является таблица 5×5, содержающая 25 символов латинского алфавита. Поскольку букв 26, обычно I и J объединяются (J либо опускается, либо заменяется на I). Каждая буква однозначно определяется парой координат (строка, столбец), где строки и столбцы нумеруются от 1 до 5. ! Квадрат может быть заполнен по ключевому слову (удаляются повторы, оставшиеся буквы добавляются по алфавиту). Это не меняет алгоритм, только отображение.
